3.制作过程

生物质燃料由秸秆、稻草、稻壳、花生壳、玉米芯、油茶壳、棉籽壳等以及“三剩物”经过加工产生的块状环保新能源。根据瑞典的以及欧盟的生物质颗粒分类标准，若以其中间分类值为例，则可以将生物质颗粒大致上描述为以下特性：生物质颗粒的直径一般为6~10毫米，长度为其直径的4~5倍，破碎率小于1.5%~2.0%，干基含水量小于15%，灰分含量小于2%，硫含量和氯含量均小于0.07%，氮含量小于0.5%。若使用添加剂，则应为农林产物，并且应标明使用的种类和数量。
